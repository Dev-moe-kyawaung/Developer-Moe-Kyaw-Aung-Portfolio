## 🧪 Quick Start (Local Development)

```bash
# 1. Clone the repository
git clone https://github.com/dev-moe-kyawaung/Developer-Moe-Kyaw-Aung-Portfolio.git

# 2. Navigate to project directory
cd Developer-Moe-Kyaw-Aung-Portfolio

# 3. Open index.html in your browser
# Or use Live Server extension in VS Code
```

## 🔧 Customization Guide
1. **Update Content**: Edit `index.html` for text changes
2. **Replace Images**: Update `assets/images/` folder
3. **Modify Colors**: Edit CSS variables in `style.css`
4. **Add Projects**: Duplicate project section in HTML
5. **Deploy**: Push to GitHub Pages or Netlify

## 📊 Deployment Options

### GitHub Pages (Current)
```bash
# Already deployed to: https://dev-moe-kyawaung.github.io/Developer-Moe-Kyaw-Aung-Portfolio/
```

### Netlify (Recommended)
1. Connect GitHub repo
2. Set build command: `none`
3. Set publish directory: `/`
4. Deploy! 🎉

### Vercel
```bash
npm i -g vercel
vercel --prod
```
